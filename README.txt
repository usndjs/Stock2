
Background:
The stock simulation trading system function has designed three modules: stockholder login, market center and stockholder operation. The system provides user registration and login module, quote inquiry page and buying & selling module. After the investors register successfully, they can log in to the system. In the market module center, users of stock speculation can inquire about the details of a listed company. The user can check the latest price, gain, buy price, sell price, transaction volume, turnover, lowest price, highest price, yesterday's closing, today's opening through the market query page.
